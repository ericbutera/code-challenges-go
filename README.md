# code-challenges-go

code challenges, golang edition

## pre-requisite steps

```sh
# ensures all dependencies are met
make setup
```

## quickstart

```sh
go test
```

## run in container

Not into installing things on the host? You can also run all the tests in a container.

```sh
make container-run
```
